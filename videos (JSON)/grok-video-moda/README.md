# Prompts Grok para vídeo de moda

Coleção extraída da conversa Geração de vídeo com Grok. Os prompts usam uma imagem base e priorizam identidade, roupa, cenário e movimentos humanos naturais.

## Como usar

Escolha apenas um hook ou uma sequência por geração. Cada arquivo já contém a instrução de seleção e está pronto para copiar no Grok.
