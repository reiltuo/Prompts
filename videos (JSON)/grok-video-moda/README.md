# Prompts Grok para vídeo de moda

Coleção organizada para geração de vídeo com uma imagem base. Os prompts priorizam identidade, roupa, cenário e movimentos humanos naturais.

## Como usar

Escolha apenas uma abertura, uma sequência ou um vídeo completo por geração. Cada JSON separado contém uma instrução de seleção única.
